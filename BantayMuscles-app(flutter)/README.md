# BantayMuscles — Flutter app

Flutter version of BantayMuscles (sibling of the Expo app). Nothing built here yet.

The Expo / React Native app lives in [`../fittracker-app`](../fittracker-app).

## Getting started
```bash
cd "BantayMuscles-app(flutter)"
flutter create .        # scaffold the Flutter project
flutter run             # or: flutter run -d chrome  (web)
```

## Notes
- Can target Android, iOS, and web from one Dart codebase.
- Would talk to the same Supabase project as the Expo app for the shared food catalog.
